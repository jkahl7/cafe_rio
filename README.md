# cafe_rio
demo llm chatbot for info about a fictional cafe

# overview
This project is broken down into a react client (generated using npx create-next-app@latest) so it has some next.js functionality (like SSR) that is as of yet unutilized.
The server was generated via npx express-generator and calls gemini with some json about the cafe. Cafe data located at data/cafe_rio.json.
 

# Installation 
Client - navigate to /client 
- npm i && npm run start
- client will run on port 3000

Server
- navigate to /server
- npm i && npm run start
- server will run on port 8080
